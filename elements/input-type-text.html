Input Type texte

L'élément HTML <input> est utilisé pour créer un contrôle interactif dans un formulaire web qui permet à l'utilisateur de saisir des données. L'élément HTML <input type="text"> permet d'ajouter du texte sur une seule ligne.

Exemple

Code derrière tout ça :

<input type="text" placeholder="Tape ton texte ici" autofocus>
<input type="text" placeholder="Je ne peux pas rester vide" required>
<input type="text" placeholder="Je suis désactivé" disabled>

Quelques attributs :

  • placeholder : Une indication, destinée à l'utilisateur, pour ce qui peut être saisi dans le champ.
  • required : Cet attribut indique que la valeur doit être renseignée avant que l'utilisateur puisse envoyer le formulaire.
  • autofocus : si un des champs a cet attribut, le curseur est placé dans ce champ quand on ouvre la page.
  • disabled : le champ est désactivé et aucune interaction n'est possible.
